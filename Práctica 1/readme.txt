Práctica 1 - Cifrado de Hill

Estrada Gómez César Derian
Cortez Flores Luis Enrique

Esta práctica contiene 3 clases:

Hill    - Está es la clase principal y la que codificará y decodificará
	      textos con Cifrado de Hill.

Algebra - Aquí se encuentran operaciones algebraicas como:
			- Determinante
			- Algoritmo de Euclides
			- Algoritmo de Euclides Extendido
			- Inverso Multiplo modular

Matriz  - Aquí se encuentran las operaciones necesarias para obtener la
          matriz inversa.
          La matriz inversa es necesaria para decodificar un texto con
          Cifrado de Hill.
